# IP Validator

A Python script that checks whether a given string is a valid IPv4 address.

## Description
This project verifies that:
- The IP has exactly 4 parts
- Each part is a number
- Each number is between 0 and 255

## Features
- Input validation
- Simple logic
- Networking concept

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python ip_validator.py

4. Enter an IP address

## Example

Input:
192.168.1.1

Output:
Valid IP Address
